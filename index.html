About Me: Tanner Rigg

Life is all about the path we take.

I still remember the first time that I peeled back the curtain on the software world. I jailbroke my iPhone 4 so that I could download a Gameboy Advanced emulator. I spent more time researching how to deploy the hack than I ever did actually using the application. When it was finished, I strutted around showing everyone my accomplishment. Everyone reacted as if I had unlocked some kind of secret magic. A path was laid before me, a trail leading to the goal of becoming a software engineer.

Unfortunately, like many passions of youth this budding interest in software development faded. When it came time to determine my major for university it never even crossed my mind that I should study computer science. I, like many lost young people, decided to follow the crowd and signed up for a major in Biology. At the time I thought that I wanted to be a doctor, then a business man, then a psychologist, and finally ended up as a neuroscientist. I followed the path until I found myself working at the National Institutes of Health. I had everything that I had been searching for, a respectable career path and plenty of opportunity for growth. All of this and something was still missing, I still didn’t have that feeling of elation that I felt the first time the emulator booted up on my phone. My last year at the NIH I was put in charge of a new project that integrated a Machine Learning Algorithm to track micro behaviors of lab mice. This reawakened my interest in software development led me to a very important conclusion about my true path in life.

There is no true path, but only the path we choose to walk. After years of prodding interest, months of studying and hard work, I have finally found myself here at beginning of my programming journey. Where will this path lead me? Who knows, but lets follow it to the end.

Foodomizer

Project 3

About Project

Attention: This project is hosted on Heroku, it may take some time for the demo to spin up.

The Foodomizer is a random meal generator designed to ease the anxiety of what to cook for breakfast, lunch, or dinner. Input the ingredients that are available, tell it what kind of recipes can be cooked and the website takes care of the rest. By checking through the ingredients available against the recipe requirements the application tells the user what recipes can be cooked and gives a random suggestion for the upcoming meal. This version of the Foodomizer was created using the Flask framework for Python and deployed on Heroku.

Plans for future improvements:
- Design new version using the Django framework that utilizes a SQL database.
- Implement more ingredient options that users can search through and add to their ingredients list more effectively.
- Introduce images for ingredients and recipes as well as adding a section for recipe instructions.
